# dndsheet Project

## Purpose
To develop a web-based application for managing Dungeons & Dragons character sheets, providing a user-friendly interface for character creation, data storage, and gameplay tracking.

## Scope
In scope: Character creation, editing, and viewing; basic combat tracking; inventory management. Out of scope: Complex campaign management, multiplayer features.

## Current Objective
To set up the project structure, establish core functionalities for character data handling, and implement a basic web interface.

## Constraints
- Python Flask framework for the backend.
- HTML/CSS/JavaScript for the frontend.
- Data persistence via local files or a simple database (e.g., SQLite).

## Expected Deliverables
- Functional web application for D&D character sheet management.
- `README.md`, `LOG.md`, and `BACKLOG.md` maintained according to project rules.
